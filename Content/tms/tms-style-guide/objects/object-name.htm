Object Name

Summary

  1. Describes the physical form, function, or medium of a work (e.g., Sculpture, Storage jar, Painting, Etching). This should be more general than the Object Title, yet more specific than Classification.
  2. Corresponds to CCO Work Type field.
  3. Use the AAT to help define terms.

 

Formatting

Uppercase the first word in the field and any proper nouns; lowercase all other words.

 

General Guidelines and Examples

Examples:

Title: Virgin and Child Enthroned with Saints Leonard and Peter

Object Name: Altarpiece

Title: Snake-Handled Trivet

Object Name: Trivet
